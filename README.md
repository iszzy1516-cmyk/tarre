# TAREÉ JEWELRY

A luxury Nigerian jewelry e-commerce platform built with **FastAPI** (backend) and **React + Vite + TypeScript** (frontend).

## Project Structure

```
TARRE/
├── taree-web/          # React 19 + Vite + TypeScript frontend
│   ├── src/
│   │   ├── components/ # UI components (layout, sections, product, cart)
│   │   ├── pages/      # Route pages
│   │   ├── stores/     # Zustand state management
│   │   ├── lib/        # Utilities & API client
│   │   └── types/      # TypeScript definitions
│   └── public/images/  # Product & brand assets
├── assets_images/      # Source product photography
├── stitch_tare_luxury_digital_boutique/  # Google Stitch design exports
└── prompt(2).md       # Full project specification
```

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | React 19 + Vite + TypeScript |
| **Styling** | Tailwind CSS v4 + Framer Motion |
| **State** | Zustand + TanStack Query v5 |
| **Routing** | React Router v7 |
| **Backend** | FastAPI + Pydantic v2 + SQLAlchemy 2.0 |
| **Database** | PostgreSQL 15+ |
| **Payments** | Paystack + Flutterwave |

## Design System

- **Deep Navy**: `#1a1f2e` — primary backgrounds
- **Champagne Gold**: `#d4af37` — accents, CTAs, headings
- **Soft Cream**: `#f5f0e8` — secondary backgrounds
- **Typography**: Playfair Display (headings) + Manrope (body)

## Getting Started

```bash
cd taree-web
npm install
npm run dev
```

The development server runs at `http://localhost:5173/`.
